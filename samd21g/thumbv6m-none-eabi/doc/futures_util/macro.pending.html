Macro pending

macro_rules! pending {

}
Expand description

A macro which yields to the event loop once.

This is equivalent to returning Poll::Pending from a Future::poll implementation. Similarly, when using this macro, it must be ensured that wake is called somewhere when further progress can be made.

This macro is only usable inside of async functions, closures, and blocks. It is also gated behind the async-await feature of this library, which is activated by default.
