1//! Module defining or exporting peripheral types for the ['ahb'], ['apb'] and
2//! ['pclk'] modules
3//!
4//! The `ahb`, `apb` and `pclk` modules each define structs that are
5//! generic over a type parameter representing a peripheral. Some peripheral
6//! modules already define suitable types for this purpose. For example,
7//! [`sercom`] defines the [`Sercom0`], [`Sercom1`], etc. types. But other
8//! peripherals are either not yet implemented in the HAL or do not define a
9//! suitable type. This module defines a type for such peripherals. If/when a
10//! suitable type is added for a given peripheral, the type defined here should
11//! be deprecated or removed.
12//!
13//! [`ahb`]: super::ahb
14//! [`apb`]: super::apb
15//! [`pclk`]: super::pclk
16//! [`sercom`]: crate::sercom
17//! [`Sercom0`]: crate::sercom::Sercom0
18//! [`Sercom1`]: crate::sercom::Sercom1
19

48macro_rules! create_types {
49    (
50        $(
51            $Type:ident
52        ),+
53    ) => {
54        $(
55            /// Marker type representing the corresponding peripheral
56            ///
57            /// This type is defined by and used within the [`clock`](super)
58            /// module. See the the [`types`](self) module documentation for
59            /// more details.
60            pub enum $Type {}
61            impl Sealed for $Type {}
62        )+
63    };
64}
