Expand description

Value-level enum of possible clock sources for a Pclk

The variants of this enum identify the Gclk used as a clock source for a given Pclk. Because the variants are identical to DynGclkId, we simply define it as a type alias.

DynPclkSourceId is the value-level equivalent of PclkSourceId.
