Expand description

Trait implemented for numeric types which can be used with the StreamCipherSeek trait.

This trait is implemented for primitive numeric types, i.e. i/u8, u16, u32, u64, u128, usize, and i32. It is not intended to be implemented in third-party crates.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
