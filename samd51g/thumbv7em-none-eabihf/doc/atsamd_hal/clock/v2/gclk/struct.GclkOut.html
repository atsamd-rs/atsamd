pub struct GclkOut<I: GclkIo> { /* private fields */ }
Expand description

A GPIO Pin configured as a Gclk output

The existence of this struct serves as proof that the corresponding Gclk is Enabled and that it has been output to PinId I.

See the module-level documentation for an example of creating a GclkOut from an EnabledGclk.
