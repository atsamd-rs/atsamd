The Enabled Xosc32k clock

As described in the clock module documentation, the Enabled wrapper implements compile-time clock tree safety by tracking the number of clocks consuming the Xosc32k clock and restricts access to the underlying type to prevent misuse.

As with Enabled, the default value for N is U0; if left unspecified, the counter is assumed to be zero.
