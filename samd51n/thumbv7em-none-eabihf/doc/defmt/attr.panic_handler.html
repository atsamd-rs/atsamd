Attribute Macro panic_handler

#[panic_handler]
Expand description

Overrides the panicking behavior of defmt::panic!

By default, defmt::panic! calls core::panic! after logging the panic message using defmt. This can result in the panic message being printed twice in some cases. To avoid that issue use this macro. See the manual for details.
